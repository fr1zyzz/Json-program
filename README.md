###  Beschreibung

Dieses Projekt ist eine selbst entwickelte Konsolenanwendung in C#, die zur Verarbeitung und Verwaltung von Daten im JSON-Format dient.
Das Projekt wurde erstellt, um grundlegende Programmierkenntnisse praktisch anzuwenden und den Umgang mit Dateien, Datenstrukturen und Programmabläufen zu vertiefen.

### Funktionen

Laden von Daten aus einer JSON-Datei

Speichern von Daten in einer JSON-Datei

Anzeigen aller Einträge

Suchen von Einträgen

Ändern des Status einzelner Einträge

### Technologien

C#

.NET

JSON

GitHub

### Ziel des Projekts

Das Ziel dieses Projekts ist es, praktische Erfahrungen in der Softwareentwicklung zu sammeln und die Grundlagen der objektorientierten Programmierung sowie der Dateiverarbeitung zu erlernen.
Das Projekt ist Teil meiner Vorbereitung auf eine Ausbildung im IT-Bereich.

### Projektstruktur

Json-program/
├── Program.cs – Einstiegspunkt der Anwendung und Menülogik und Benutzereingaben  
│
├── Managmant/ – Verwaltung der Daten sowie Laden und Speichern von JSON-Dateien  
│
├── SearchingCompany/ – Suchfunktionen innerhalb der Daten  
│
├── ChangeStatus/ – Ändern des Status einzelner Einträge  
│
├── DeleteFromJson/ – Löschen von Einträgen aus der JSON-Datei  
│
├── ShowList/ – Anzeigen aller gespeicherten Einträge  
│
├── User/ – Datenmodelle (z. B. Application)  
│
├── users.json – JSON-Datei mit gespeicherten Daten  
├── Json-program.csproj – .NET-Projektkonfiguration  
└── README.md – Projektbeschreibung



### Описание

Этот проект — самостоятельно разработанное консольное приложение на C# для обработки и управления данными в формате JSON.
Проект был создан с целью практического применения базовых знаний программирования и изучения работы с файлами, структурами данных и логикой программы.
